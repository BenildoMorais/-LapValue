Introdução

Com a expansão dos mercados digitais, qualquer pessoa pode vender online, publicando facilmente seus produtos em diversas plataformas. No entanto, muitos vendedores, especialmente os que não são profissionais, enfrentam um desafio comum: definir o preço correto, especialmente quando se trata de laptops. Os preços desses dispositivos podem variar amplamente de acordo com suas especificações, tornando difícil a avaliação precisa.

Felizmente, o avanço do aprendizado de máquina oferece uma solução. Com modelos treinados em dados reais de laptops, é possível prever o preço ideal com base nas características do produto. Este projeto tem como objetivo criar e treinar um modelo de aprendizado de máquina que, utilizando um dataset composto de especificações e preços de laptops, consiga prever o valor de mercado mais justo para cada equipamento.

DataSet

Entitulado "Laptop Prices" é um Dataset que pode ser encontrado no site da kaggle disponibilisado em setembro de 2024 sendo um conjunto de dados sobre especificações de diferentes modelos de laptops onde incluia 13 atributos das suas caracteristicas que são listadas abaixo

  • Marca
  • Producto
  • Nome do Modelo
  • Tamanho da Tela
  • Resoluçao da Tela
  • Processador
  • Memoria Ram
  • Armazenamento
  • Placa Grafica
  • Sistema Operativo
  • Peso
  • Preço

Solução

Solução do problema de Pesquisa

Regressão Linear Simpples

O que é regressão linear simples?

A regressão linear simples é uma metodologia que estuda a relação entre dois fenômenos, permitindo entender o efeito e a causalidade entre eles, além de prever novos valores. Para compreender melhor, vamos desvendar como ajustar a reta, interpretar e prever valores.

Caracteristicas
  • Dependencia entre duas variaveis
  • Basea-se de uma equacção Linear
  • É um algoritmo de aprendizado supervisionado

Como Foi implementado

Para o presente projecto foi criado uma API em java atravez da framework Spring onde o modelo treinado com o dataset da lista de laptops e seus preços

A API Recebe um Método post com os atributos do laptop e atravez do modelo criando faz a predição e retorna o preço previsto

Interfaçe do utilizador

Baseado em HTML, CSS, JavaSCript e bootstrap a interface do utilizador foi feita com base num template de pórtifólio que foi alterado para poder fornecer explicação sobre o problema e a descrição do problema e por sua vez um ambiente de teste que é responsavel por receber as caracteristicas dos laptops e por sua vez enviar os dados a API que faz o resto da operação e devolve o preço
